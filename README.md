# Анализ курсов валют

Этот проект представляет собой простой пример исследовательского анализа данных (EDA) для курсов валют.

## Использованные технологии

*   Python
*   Pandas
*   Matplotlib
*   Seaborn
*   Jupyter Notebook

## Данные

Искусственно сгенерированные данные курса USD/RUB за последние 30 дней.

## Запуск

1.  Скачайте файл `currency_analysis.ipynb`.
2.  Откройте его в Jupyter Notebook (например, через Google Colab).
3.  Выполните все ячейки.
